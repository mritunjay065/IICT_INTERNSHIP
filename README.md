# IICT Internship Projects

This repository contains the projects completed during the IICT Internship.

## Projects Directory

*   **[Project 1: AI-Powered Fake News Detection Using Text Classification](./project1)**: A machine learning pipeline classifying news articles as real or fake using custom preprocessing, TF-IDF vectorization, and model training (KNN, Logistic Regression, Random Forest, Neural Network).
*   **[Project 2: AI-Driven Phishing Email & Fake News Detection Suite](./project2)**: A unified security dashboard detecting phishing email threats (using text and structural metadata) and analyzing live global news on the fly using NewsAPI.
